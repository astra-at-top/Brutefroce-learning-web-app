## Workflow 1: Export вҶ’ Modify вҶ’ Reimport

Use this when you want to take existing data, modify it with an AI, and bring it back.

```
в”Ңв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”җ     в”Ңв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”җ     в”Ңв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”җ     в”Ңв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”җ
в”Ӯ  Admin   в”Ӯ     в”Ӯ  Export  в”Ӯ     в”Ӯ    AI     в”Ӯ     в”Ӯ  Import  в”Ӯ
в”Ӯ  Tab     в”Ӯ в”Җв”Җв–¶ в”Ӯ  Modal   в”Ӯ в”Җв”Җв–¶ в”Ӯ  (ChatGPT в”Ӯ в”Җв”Җв–¶ в”Ӯ  Modal   в”Ӯ
в”Ӯ          в”Ӯ     в”Ӯ          в”Ӯ     в”Ӯ  /Claude) в”Ӯ     в”Ӯ          в”Ӯ
в”Ӯ  Click   в”Ӯ     в”Ӯ  Click   в”Ӯ     в”Ӯ           в”Ӯ     в”Ӯ  Click   в”Ӯ
в”Ӯ  рҹ“ӨBtn   в”Ӯ     в”Ӯ  "Export"в”Ӯ     в”Ӯ  Modify   в”Ӯ     в”Ӯ "Import" в”Ӯ
в””в”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”ҳ     в””в”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”ҳ     в””в”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”ҳ     в””в”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”ҳ
```

### Step-by-step:

**Step 1 вҖ” Export current data:**
1. Go to **Admin** tab
2. Click **рҹ“Ө Export JSON** button in the editor header (right side)
3. A modal opens вҖ” click **в–¶ Export as JSON**
4. The textarea fills with all courses, lessons, quizzes, coding exercises, and flashcards
5. Click **рҹ“Ӣ Copy** to copy the JSON to clipboard

**Step 2 вҖ” Modify with AI:**
1. Paste the JSON into ChatGPT, Claude, or any AI
2. Tell the AI what to change вҖ” e.g.:
   - "Add a new lesson about closures to the JavaScript course"
   - "Update the quiz for vars-types with harder questions"
   - "Add flashcards to the arrays-objects lesson"
3. AI returns modified JSON вҖ” copy it

**Step 3 вҖ” Import back:**
1. Back in Admin tab, click **рҹ“Ҙ Import JSON** button
2. Paste the modified JSON into the textarea
3. Select mode:
   - **Import Courses (merge)** вҖ” for full course+lesson imports
   - **Import Single Lesson** вҖ” for just one lesson (enter Course ID)
4. Click **в–¶ Import JSON**
5. Modal closes automatically on success
6. Admin tree refreshes with new/updated data

## Workflow 2: Create Fresh Content from AI

When you have AI generate brand-new content (no export needed):

### Step-by-step:

1. Ask the AI: *"Generate JSON for a JavaScript course with 3 lessons including quizzes and flashcards. Use this format:"*
2. Copy the JSON format from the placeholder in the import modal (it's pre-filled)
3. Paste AI's output into the **рҹ“Ҙ Import JSON** modal
4. Click **в–¶ Import JSON**
5. New courses/lessons appear in the Content Tree

## JSON Format

### Full Course Import

```json
{
  "courses": [
    {
      "id": "course-id",
      "name": "Course Name",
      "lessons": [
        {
          "id": "lesson-id",
          "title": "Lesson Title",
          "status": "available",
          "type": "lesson",
          "notes": "Lesson notes here...",
          "mnemonic": "A memory trick",
          "quiz": [
            {
              "question": "What is X?",
              "options": ["A", "B", "C", "D"],
              "correct_index": 0
            }
          ],
          "coding": {
            "prompt": "Write a function...",
            "starter": "// your code here",
            "solution": "function solve() { return 42; }"
          },
          "flashcards": [
            { "front": "Term", "back": "Definition" }
          ]
        }
      ]
    }
  ]
}
```

### Single Lesson Import

```json
{
  "lesson": {
    "id": "my-lesson",
    "title": "My Lesson",
    "status": "available",
    "type": "lesson",
    "notes": "...",
    "quiz": [...],
    "flashcards": [...]
  },
  "course_id": "target-course-id"
}
```

## Important Notes

- **Merge mode** вҖ” always on. Existing courses/lessons with matching IDs get updated; new ones are added; anything not in the JSON stays untouched
- **Progress preserved** вҖ” user progress on lessons is NOT overwritten
- **Flashcard reviews preserved** вҖ” spaced-repetition data stays intact
- **IDs matter** вҖ” to update an existing lesson, use the same `id` as the current one
- **Course must exist** вҖ” for single lesson import, the target course must already exist

## Tips for AI

When asking an AI to generate content, include the expected format in your prompt:

> *"Generate learning content in this JSON format for import into a learning app: [paste the format above]. Create 3 lessons about [topic] with 2 quiz questions each and 2 flashcards each."*

The AI will follow the structure and you can import directly.
